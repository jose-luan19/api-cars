# API-Cars
## Execução via Docker

```bash
docker-compose up
```
## Documentação da API com Swagger
Com a aplicação sendo executada na porta 8080, acesse a url:

http://localhost:8080/swagger-ui.html

Com essa interface poderá visualizar e testar os endpoints criados
